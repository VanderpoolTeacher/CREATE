# Curriculum — Turnkey Units

Five pick-up-and-teach units, one for each durable skill, designed to accompany the talk *[Rebuilding the Foundations for Student Success](../presentation/outline.md)*. Each unit is a complete lesson plan: objective, materials, a timed session-by-session sequence, a copy-ready student handout, teacher look-fors, the matching rubric, a gradual-release plan, differentiation notes, and common pitfalls.

> **These units are optional — and that's by design.** The primary path is the [weave-in strategies](../toolkit/weave-in-strategies.md): small tweaks to assignments you already give, no new course required. These units are for when you have room to carve out dedicated practice time. Every unit ends with a **"low-lift version"** — the same skill compressed into a one-sentence weave-in move — so you can start small and grow into the full unit when the timing is right.

## How to use these

1. **Pick one skill your students most need.** Not sure? Start with **The Ambiguous Brief** — Working with Ambiguity and Decomposition sit underneath everything else.
2. **Read the unit once, end to end.** Each is built to run with minimal prep after the first time.
3. **Don't run the sessions back-to-back.** Most units are designed to be *spaced* across a term so the scaffolding can actually withdraw — that gradual release is the point, not a nicety.
4. **Assess with the rubric, not a new grade.** Each unit names the exact competency rows from [`toolkit/rubrics.md`](../toolkit/rubrics.md) to lay over the work students already produce.
5. **No room for the full unit?** Use the **low-lift version** at the bottom of each one. It's the entire unit compressed into a single weave-in move.

## A note on grade bands

The units are written for **grades 6–12** as the default, with explicit adaptation notes for **grades 3–5** and **college / CTE** in every file. Scale the briefs, drop the time estimates for younger students, and raise the stakes (real clients, real deadlines) for older ones — the underlying skill is the same at every age.
